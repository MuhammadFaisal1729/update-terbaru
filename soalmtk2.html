Contoh Soal Turunan

Soal 1

Hitung turunan fungsi: f(x) = 4x^3 + 7x^2 - 5x + 8
f'(x) = 12x^2 + 14x - 5 Penjelasan langkah: - Turunan dari 4x³ adalah 4 × 3 x² = 12x² - Turunan dari 7x² adalah 7 × 2 x¹ = 14x - Turunan dari -5x adalah -5 - Turunan dari 8 (konstanta) adalah 0 Jadi, f'(x) = 12x² + 14x - 5

Soal 2

Turunkan fungsi berikut: g(x) = 5x^4 - 3x^3 + x - 10
g'(x) = 20x^3 - 9x^2 + 1 Langkah-langkah: - Turunan 5x⁴ adalah 5 × 4 x³ = 20x³ - Turunan -3x³ adalah -3 × 3 x² = -9x² - Turunan x adalah 1 - Turunan -10 adalah 0 Jadi, g'(x) = 20x³ - 9x² + 1

Soal 3

Hitung turunan fungsi: h(x) = (3x^2 + 1)(x^3 - 4x)
Gunakan aturan produk: h'(x) = f'(x)g(x) + f(x)g'(x) Misal: f(x) = 3x^2 + 1 → f'(x) = 6x g(x) = x^3 - 4x → g'(x) = 3x^2 - 4 Jadi: h'(x) = (6x)(x^3 -4x) + (3x^2 + 1)(3x^2 - 4) = 6x^4 - 24x^2 + 9x^4 - 12x^2 + 3x^2 - 4 = (6x^4 + 9x^4) + (-24x^2 -12x^2 + 3x^2) - 4 = 15x^4 - 33x^2 -4

Soal 4

Turunkan fungsi: j(x) = \(\frac{2x^3 - 5}{x^2 + 1}\)
Gunakan aturan pembagian: j'(x) = \(\frac{f'(x)g(x) - f(x)g'(x)}{(g(x))^2}\) Misal: f(x) = 2x^3 - 5 → f'(x) = 6x^2 g(x) = x^2 + 1 → g'(x) = 2x Jadi: j'(x) = (6x^2)(x^2 + 1) - (2x^3 - 5)(2x) / (x^2 + 1)^2 = (6x^4 + 6x^2) - (4x^4 - 10x) / (x^2 + 1)^2 = (6x^4 + 6x^2 - 4x^4 + 10x) / (x^2 + 1)^2 = (2x^4 + 6x^2 + 10x) / (x^2 + 1)^2

Soal 5

Hitung turunan fungsi: k(x) = e^x + 3x^5
Turunan fungsi eksponensial dan pangkat: k'(x) = e^x + 15x^4 Penjelasan: - Turunan dari e^x adalah e^x - Turunan dari 3x^5 adalah 15x^4 (3 × 5 x^4)
